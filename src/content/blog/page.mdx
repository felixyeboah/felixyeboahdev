# MDX Setup Complete

This route is rendered from an `.mdx` file and uses custom components from `mdx-components.tsx`.

<Callout title="What you get" tone="success">
  GitHub-style syntax highlighting, polished typography defaults, and a copy button for every fenced code block.
</Callout>

## Useful components

- `Callout` with `info`, `success`, and `warning` tones.
- Styled tables, blockquotes, inline code, and headings.
- External links open in a new tab.

## Code block with copy

import { HugeiconsIcon } from "@hugeicons/react";
